# Battleship Game Simulation

## Description
A program based on the rules of the Battleship board game randomly places ships on two boards and simulates gameplay between 2 players.

## DEMO 
https://mikolajgraczyk.github.io/battleship-game-simulation/

## Technologies used in project
- html
- css
- JSX
- React
- styledComponents
- TypeScript
- SVG images
